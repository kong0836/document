# 不能clean之前生成的jar包

[ERROR] Failed to execute goal org.apache.maven.plugins:maven-clean-plugin:3.1.0:clean (default-clean) on project mis-dingtalk-api: Failed to clean project: Failed to delete /var/lib/jenkins/workspace/mis-dingtalk/mis-dingtalk-api/target/mis-dingtalk-api-1.0-SNAPSHOT.jar -> [Help 1]

  • 修改jenkins默认用户为root

    $ vim /etc/sysconfig/jenkins

# 将原来的jenkins用户修改为 root用户
JENKINS_USER="root"

  • 更改jenkins三个文件夹的用户所有都和用户组

    $ chown -R root:root /var/lib/jenkins
$ chown -R root:root /var/cache/jenkins
$ chown -R root:root /var/log/jenkins

  • 重启Jenkins服务并检查运行Jenkins的用户是否已经切换为root

    $ systemctl restart jenkins

  • 查看Jenkins进程所属用户

    $ ps -ef | grep jenkins
