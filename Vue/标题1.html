# 全局filters

  • 在vue项目的 src 目录下新建一个文件夹 plugin，然后在文件夹下新建一个 filters.js 文件，文件内容参考如下代码：

    // 需要安装Moment.js
// npm install moment --save 
import moment from "moment";

/**
 * 全局filters
 */
export default {
    /**
     * 日期格式化
     * @param value
     * @returns {string}
     */
    dateFilter(value) {
        if (!value) {
            return '暂无';
        }
        return moment(value).format('YYYY-MM-DD');
    },

    /**
     * 日期格式化
     * @param value
     * @returns {string}
     */
    dateTimeFilter(value) {
        if (!value) {
            return '暂无';
        }
        return moment(value).format('YYYY-MM-DD HH:mm:ss');
    },
}

  • 在vue项目下的 main.js 中，写入如下代码：

    import filters from "@/plugins/filters";

// 全局导入过滤器,必须写在 new Vue()之前，否则会不生效！
Object.keys(filters).forEach(key => Vue.filter(key, filters[key]));

  • 在组件中正常使用即可生效

# 事件修饰符

# stop

​ 等同于 JavaScript 中的 event.stopPropagation()，防止事件冒泡。

# self

​ 只会触发自己范围内的事件，不包含子元素。

# prevent

​ 等同于JavaScript中的event.preventDefault()，防止执行预设的行为（如果事件可取消，则取消该事件，而不停止事件的进一步传播）。

# once

​ 只会触发一次。

# passive

​ 告诉浏览器不阻止任何默认行为。
